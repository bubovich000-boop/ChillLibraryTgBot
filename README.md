## ⚙️ Troubleshooting

- **Bot Does Not Respond:**  
  Check your internet connection and Telegram app. Restart the bot by typing /start again.  
- **Downloads Not Working:**  
  Make sure your browser settings allow downloads. Try a different browser if needed.  
- **Photo Search Not Accurate:**  
  Use a clear, well-lit photo focusing on the book cover’s text. Avoid blurry or dark images.  
- **Quote Search Not Showing Results:**  
  Try sending a longer or more specific quote.
